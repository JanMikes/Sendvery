# Background Pattern — Subtle Geometric Grid

**Save as:** `assets/images/bg-pattern.png`
**Dimensions:** 800x800 (seamless tile)
**Use:** Repeating section backgrounds on landing page and tool pages. Applied via CSS at 3-8% opacity.

---

Create a high-quality, ultra-sharp, crisp image at maximum resolution. Every edge must be clean and well-defined. No blur, no softness, no compression artifacts. Pixel-perfect rendering.

Generate an image with the following specifications. Follow every detail precisely.

**Subject:** A seamless, tileable geometric pattern for a tech SaaS website background. The pattern consists of a hexagonal grid with thin connection lines between nodes. Some intersection points have small dots (3-4px feel), others have tiny abstract shapes — miniature shields or envelope outlines.

**Exact colors:**
- All pattern elements: single color, teal #0F766E
- Background: pure white #FFFFFF (it will be inverted/opacity-adjusted in CSS)

**Composition:**
- Exactly 800x800 pixels, square
- The pattern MUST tile seamlessly — edges must match perfectly on all four sides
- Even distribution across the entire tile
- Consistent line weight throughout (very thin, 1px feel)
- Nodes spaced evenly, approximately 80-100px apart

**Style rules:**
- Ultra-minimal — the thinnest possible lines
- This pattern will be used at 3-8% opacity, so it needs to be subtle even at full opacity
- Single color only — no gradients, no fills, no color variation
- Geometric precision — clean angles, consistent spacing
- The pattern should feel technical and structured, like a circuit board or network topology
- No text, no logos, no recognizable shapes beyond simple geometric forms

**What to avoid:**
- Thick lines or heavy shapes — this must be airy and light
- Irregular spacing or randomness — keep it structured
- Multiple colors or gradients
- Organic shapes — everything geometric
- Patterns that create visual "hot spots" when tiled (uneven density)
- Blurry, soft, or out-of-focus rendering — every edge and line must be razor-sharp and crisp

**Critical:** Test the tiling — the left edge must connect cleanly to the right edge, and the top edge must connect cleanly to the bottom edge when repeated.
