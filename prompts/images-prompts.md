# Sendvery — Image Generation Prompts (ChatGPT / DALL-E)

## Style Guide for ALL Images

**Every prompt below should produce images that share this visual language:**

- **Color palette:** Deep teal (#0F766E) as primary, indigo (#4338CA) as accent, warm amber (#F59E0B) for highlights, dark slate (#0F172A) for backgrounds. White and light gray for contrast.
- **Style:** Clean, modern, geometric, slightly abstract. Soft gradients. Glass morphism elements. NO cartoonish illustrations, NO stock photo realism, NO flat clip art.
- **Aesthetic references:** Linear.app, Resend.com, Raycast.com — premium, developer-oriented, confident.
- **Mood:** Professional but approachable, technical, trustworthy, modern.
- **Rendering:** Smooth 3D-ish renders or clean geometric compositions. Soft lighting, subtle shadows. Think polished product design, not illustration.
- **Background:** Dark (#0F172A slate) or transparent where noted. Marketing site is dark-mode-first.
- **Aspect ratios are specified per image. Follow them exactly.**

1. **Hero background** (#1) — defines the entire visual language
2. **OG image** (#5) — needed before any page goes live
3. **Background pattern** (#6) — used on every page

## TIPS FOR GENERATION

- Generate each prompt 2-3 times and pick the best variation
- If the colors aren't matching, add "Exact hex colors:" followed by the palette
- For the dark background images, you may need to say "pure dark background, no lighter elements behind the main subject"
- If images are too busy, add "extremely minimal, only the key elements, lots of negative space"
- If the 3D style isn't right, try replacing "3D" with "2.5D isometric" or "flat geometric with subtle depth"
- Save originals at max resolution — you can always scale down
- Name files descriptively: `hero-bg-v1.png`, `how-it-works-connect-v2.png`, etc.
